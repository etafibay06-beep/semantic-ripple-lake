# Consciousness Dynamical System - Experimental Framework

## 1. Phase Diagram Scan

Vary parameters $(\gamma, \lambda)$ and measure:
- Mean branching factor $\langle b \rangle$
- Normalized entropy $\tilde{H}$
- Spike rate
- Stability/failure rate

## 2. Critical Window Performance

Compare retrieval diversity across different $b$ regimes:
- Subcritical ($b < 1$)
- Critical ($b \approx 1$)
- Supercritical ($b > 1$)

Metrics:
- Result diversity
- Novelty score
- User satisfaction

## 3. Avalanche Distribution

Measure spike avalanche:
- Size distribution $P(s)$
- Duration distribution $P(T)$

Expected: Power-law near criticality

## 4. Consciousness State Transitions

Track state transitions:
- Silent → Drowsy → Active → Awake
- Measure transition probabilities
- Validate hysteresis

## 5. System Robustness

- Energy monitoring
- Backtracking success rate
- Adaptive step size effectiveness
